Руководство по запуску.

Visual Studio
При выполнении следующих пунктов обратите внимание, что в некоторых местах нужно писать Permutations, а в других Permutation
1) Создайте новый проект и назовите его Permutations во избежание проблем со сборкой
2) В создавшемся проекте кликните ПКМ в обозревателе решений по строке с названием Permutations (НЕ решение "Permutations"). В выбранном меню нажмите "добавить" --> "класс".
3) Последовательно добавьте классы с названиями  Permutation.cs, Cycle.cs, Util.cs. (Program.cs уже должен быть создан)
4) Удалите весь код из созданных файлов и замените его кодом из соответствующих файлов в GitHub (Programs.cs тоже, там вас ждет небольшой туториал)
5) Запускайте проект, изучайте, развлекайтесь


Для счастливых обладателей Rider
1) Создаете новое консольное приложение, называете проект Permutations
2) Слева есть вкладка с иерархией проекта. В ней нажимаете ПКМ по вкладке Permutations (на ней зеленым цветом написано C#), нажмите Add --> Class/Interface
3) Последовательно добавьте классы с названиями  Permutation.cs, Cycle.cs, Util.cs. (Program.cs уже должен быть создан)
4) Удалите весь код из созданных файлов и замените его кодом из соответствующих файлов в GitHub (Programs.cs тоже, там вас ждет небольшой туториал)
5) Запускайте проект, изучайте, развлекайтесь
